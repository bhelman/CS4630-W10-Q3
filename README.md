# CS4630-W10-Q3
W10 Q3 Learning Activity for CS 4630

## Setup:
1. Install Python libraries.
   * Windows 11: `py -m pip install pandas numpy scikit-learn imbalanced-learn`
2. Either download `data.csv` from the repo, **OR** download the data set from its original source at: https://archive.ics.uci.edu/dataset/697/predict+students+dropout+and+academic+success
3. Ensure `data.csv` and `student_info.py` are in the same folder.
4. Run Python file `student_info.py` and observe console output.
